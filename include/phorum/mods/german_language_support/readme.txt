German Language Support Module (5.1 and 5.2 ready)

Provides homepage specific text replacing text from core and module language
files, alternative terms for some key terms and an extended format for
notifications emails. Only German language support!

Tested on version 5.1.17 and future 5.2 (CVS till revision 1468)

Installation:
------------
This module is part of the German Language Files Package. See installation
instructions in: docs/language-german-readme.txt.

After installation of the German Language Files Package:

1. Optional: Login as administrator in Phorum´s administrative interface and go
   to the "Modules" section.

2. Optional: Enable the "German Language Support Module".

Module settings:
---------------
1. Select translation for "Buddy"

Select one of the offered translations for "Buddy": Buddy, Freund/Freundin,
Kollege/Kollegin, Kumpel/Bekannte.

The default value is "Freund/Freundin".

2. Select translation for "Sticky"

Select one of the offered translations for "Sticky": Festgepinnt, Pickert
[österr.][ugs.], Sticky, Wichtig.

The default value is "Wichtig".

3. Format of notification emails

Select a format for the notification emails send to users. Standard format
provides only a link to the new topic. Extended format includes additional the
topic text.

The default value is "Extended".

Language support:
----------------
This module supports only German language.

Sanity Checks:
-------------
If you have problems with this module login as administrator in Phorum´s
administrative interface and go to the "System Sanity Checks" section. Check
for warnings or errors for the German Language Support Module.
